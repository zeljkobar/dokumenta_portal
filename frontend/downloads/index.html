Desktop Helper Downloads

Build artefakti ce se pojaviti ovdje nakon lokalnog ili CI build-a.

Pokreni build u folderu desktop-helper/:

  • npm run build:mac za .dmg
  • npm run build:win za .exe

Output je podešen na frontend/downloads/.

Download za Windows (.exe) Download za macOS (.dmg) Nazad na Admin

macOS: ako javlja "is damaged"

Za unsigned app (bez Apple Developer naloga), pokreni ove 2 komande u Terminalu: 

xattr -dr com.apple.quarantine ~/Downloads/Dokumenta\ Helper-1.0.0-arm64.dmg
xattr -dr com.apple.quarantine /Applications/Dokumenta\ Helper.app && open /Applications/Dokumenta\ Helper.app

Windows varijanta

Windows installer (.exe) je dostupan za preuzimanje iznad. Novi build se pravi komandom npm run build:win u folderu desktop-helper/.
